## 📌 About the Project
CodevOS 2.0 is a browser-based desktop simulation built with HTML, CSS, and JavaScript. It features draggable windows, a taskbar, built-in apps like calculator, files, and browser, plus a login system and customizable wallpapers—all running fully on the client side.

## ⚙️ Features
CodevOS 2.0 is a browser-based desktop environment with draggable windows, taskbar, and a modern UI.
It includes built-in apps like calculator, calendar, clock, file explorer, browser, and settings.
Features login system, customizable wallpapers, and smooth animations—all running fully client-side.
